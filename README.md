# uso-ollama-01

1. Para ejecutar run_local.py; usted tener levantado el proceso de ollama, para este ejemplo se usa el "model": "llama3.2:3b",

## Comandos ollama

1. Descargar Ollama: https://ollama.com/ (para su sistema operativo)

2. Desde CMD (Windows) o Terminal (Linux / Mac) 
2.1. Verificar la versión
* ollama -v
* ollama version

2.2. Obtener un LLM liviano

* ollama pull llama3.2:3b

2.3. Comprobar que esté el servicio de ollama activado

* En un navegador, comprobar que esté activo con la siguiente dirección http://localhost:11434/
