# uso-ollama-01

1. Para ejecutar run_local.py; usted tener levantado el proceso de ollama, para este ejemplo se usa el "model": "llama3.2:3b",

2. Para ejecutar run_nube.py; usted debe:

* Crear una cuenta en mistral: https://admin.mistral.ai/
* Obtener una API para consumir el API. Hasta la fecha de creación de este repositorio, mistral permite crear API's gratuitas con límites.
* El archivo config.py.bk se debe copiar en un archivo llamado config.py . 
* Luego debe ingresar el API generado en mistral en config.py
